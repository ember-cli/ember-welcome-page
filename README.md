ember-welcome-page
==============================================================================

Displays a nice welcome page for newly generated applications with links to resources.


Installation
------------------------------------------------------------------------------

```sh
ember install ember-welcome-page
```


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.28 or above
* Ember CLI v3.28 or above
* Node.js v14 or above
